GOTTA.BIKE (gotta.bike) is a personal, static portfolio website. It keeps things simple: no accounts, no forms, no ads, no analytics scripts, and no cookies set by this site.

## What this site collects

Nothing directly. This site has no sign-ups and does not set or read cookies. There is no tracking code on these pages.

## Hosting and third-party services

Like nearly every website, the servers that deliver these pages may process standard technical data (such as your IP address and browser type) in their logs. This site is hosted on GitHub Pages and delivered through Cloudflare. Web fonts are loaded from Google Fonts, and CSS/JS libraries are loaded from the jsDelivr CDN. Each of these providers has its own privacy policy: [GitHub](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement), [Cloudflare](https://www.cloudflare.com/privacypolicy/), [Google](https://policies.google.com/privacy), and [jsDelivr](https://www.jsdelivr.com/terms/privacy-policy-jsdelivr-net).

## Links to other sites

This site links to external services such as GitHub and Discord, and to the individual GOTTA.BIKE project apps (for example worlds.gotta.bike or print.gotta.bike). Those sites and apps have their own privacy policies — this policy covers only this website.

## Questions

If you have any questions about privacy on this site, reach out on [GitHub](https://github.com/vincentdavis) or [Discord](https://discord.com/users/588793677317537811).
